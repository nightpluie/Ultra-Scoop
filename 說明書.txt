=====================================
  ULTRA SCOOP  —  使用說明書
=====================================


【這是什麼程式？】

ULTRA SCOOP 是一個 AI 寫稿工作站。
你可以把記者會的錄音丟進去，或是直接現場即時轉錄，
程式會根據你的寫作風格自動生成新聞稿，然後讓你查核、修改、輸出。


--------------------------------------------------------------------
一、安裝（只需做一次）
--------------------------------------------------------------------

1. 確認你的 Mac 是 macOS Monterey（12）以上（需 Apple 晶片 M1／M2／M3…）。

2. 第一次開啟「安裝（第一次執行）.command」時，
   因為檔案是從別人那裡傳來的，macOS 會擋下來，請改用：
      在該檔案上「按右鍵 →（按住 control 點一下）→ 打開 → 再按一次打開」
   只有第一個檔案需要這樣做，安裝程式會自動解除其餘檔案的限制，
   之後雙擊「啟動」「更新」都不會再被擋。

3. 看到黑色視窗跑字，過程大約 5～10 分鐘，
   期間會安裝程式所需的套件，以及下載 Whisper AI 語音模型（約 1.5GB），
   請保持網路連線，不要關掉視窗。

4. 看到「✅ 安裝完成！」就大功告成，可以關掉視窗了。

⚠️  如果出現「找不到 Python 3」：
    請打開終端機（Terminal），輸入以下指令後按 Enter，
    等安裝完畢再重跑安裝程式：
        xcode-select --install


--------------------------------------------------------------------
二、每次使用
--------------------------------------------------------------------

雙擊：
      啟動 Ultra Scoop.command

程式介面就會開啟。

第一次使用時，程式會自動開啟「⚙ 設定」視窗，
請在裡面填入 Anthropic API Key（向提供程式的人索取），
勾選「記憶 API Key」後按「完成」，之後就不用再填。


--------------------------------------------------------------------
三、主要功能說明
--------------------------------------------------------------------

【輸入素材（v3.4 改為分頁）】

「輸入素材」欄上方有四個分頁，點選切換，一次只顯示一種：
  - 現場錄音　　：用筆電麥克風即時轉錄，紅鈕開始、方鈕停止
  - 手機麥克風　：切到此分頁會自動顯示 QR Code，
                  手機掃描後在手機網頁按「開始」即可錄音（切走自動關閉）
  - 音檔轉錄　　：匯入 MP3 / WAV / M4A / 影片檔轉成逐字稿
  - 附件　　　　：上傳 PDF / Word / Excel / TXT，會連同逐字稿一起給 AI

【生成報導】

1. 用上述任一方式取得逐字稿（或直接貼上文字）。
2. 填入受訪者 / 發言人、報導角度（可留空讓 AI 自行判斷）。
3. 點「生成報導」，稿件即時串流顯示於右欄。

※ API Key、Claude 模型、技能包、Whisper 模型等設定，
   都在右上角「⚙ 設定」（快捷鍵 Cmd+,）。

【配稿模式（v3.0 新功能）】

1. 在生成報導設定欄勾選「同時產出配稿」。
2. 可填入「配稿角度」，或留空讓 AI 自行找出互補角度。
3. 生成後，右欄出現「主稿 / 配稿」Tab，點擊切換兩篇稿件。

【逐字稿修正錯字（v3.0 新功能，v3.2 強化，v3.4 更名）】

逐字稿完成後，點「修正錯字」按鈕，AI 會自動修正語音辨識的錯字
（人名、機構名、專有名詞等），不改動句子結構與格式。

修正時，程式會自動把填入的「受訪者 / 發言人」以及
本場記者會名稱一併提供給 AI 作為參考，
讓人名與專有名詞的修正更精準。建議在錄音前填好受訪者欄位。

長篇逐字稿（30 分鐘以上）不會截斷，v3.2 已改為分段處理後合併。

【事實查核（v3.0 升級，v3.4 新增點擊跳轉）】

1. 稿件生成後，點「開始查核」。
2. AI 會找出三類問題：
   - 錯字（typo）：人名、機構名拼寫錯誤
   - 數據不符（mismatch）：數字、日期與原始素材不符
   - 無出處（unsourced）：稿件內容在素材中找不到依據
3. 直接點擊查核結果中的問題文字，右欄稿件會自動捲到對應位置並亮起。
4. 點「標記問題」：右欄稿件以顏色標記（黃底 = 可自動修正、灰底 = 需人工確認）。
5. 點「一鍵修正（N 項）」：自動套用所有有建議值的修正。
   灰底標記仍需自行逐一確認。

【錄音自動存檔（v3.1 新功能，v3.4 新增自動清理）】

錄音結束後，音訊自動儲存至程式同目錄的 rec/ 資料夾，無需手動操作。

- 筆電麥克風：存為 live_日期時間.wav
- 手機麥克風：存為 phone_日期時間_記者會主題.wav
  （手機端有填入記者會名稱時，自動帶入檔名，方便辨識）

格式為標準 WAV，可直接用 QuickTime 或任何播放器開啟。
可作為逐字稿有誤時的原始音檔備份。

啟動程式時若發現超過 30 天的錄音備份，會詢問是否清理，
避免錄音檔越積越多佔滿磁碟（逐字稿文字備份在 log/，不受影響）。

--------------------------------------------------------------------
五、搬到新電腦
--------------------------------------------------------------------

1. 把整個「Ultra Scoop」資料夾複製到新 Mac（放哪裡都可以）。
2. 雙擊「安裝（第一次執行）.command」重新安裝一次。
3. 安裝完成後，之後都雙擊「啟動 Ultra Scoop.command」即可。


--------------------------------------------------------------------
六、macOS 安全性警告（第一次執行必看）
--------------------------------------------------------------------

從別人電腦收到這個資料夾時，macOS 可能會彈出警告：
「無法打開，因為 Apple 無法檢查其是否含有惡意軟體」

這是 macOS 的正常安全機制，不代表程式有問題。
按照以下步驟解除即可：

【方法一：右鍵開啟（最簡單）】
  1. 在「安裝（第一次執行）.command」上按右鍵
  2. 選「開啟」
  3. 跳出警告視窗後，再按一次「開啟」
  4. 之後就能直接雙擊了

【方法二：系統設定解除（如果右鍵沒用）】
  1. 點 Apple 選單 →「系統設定」→「隱私權與安全性」
  2. 往下滑，找到「已封鎖『安裝（第一次執行）.command』」
  3. 點「仍要開啟」→ 輸入 Mac 密碼確認
  4. 回到資料夾，雙擊即可執行

啟動程式（啟動 Ultra Scoop.command）若也出現相同警告，
用一樣的方式處理一次就好，之後不會再出現。


--------------------------------------------------------------------
七、常見問題
--------------------------------------------------------------------

Q：雙擊 .command 沒反應？
A：在 .command 檔案上按右鍵 → 選「開啟」→ 系統詢問時選「允許」。
   之後就能直接雙擊了。

Q：安裝完可以沒網路使用嗎？
A：可以。安裝時已把 AI 語音模型下載到本機，
   寫稿功能需要 Anthropic API 金鑰，那個需要網路。

Q：想換更精準的模型（例如 large-v3）怎麼辦？
A：點右上角「⚙ 設定」，在「語音轉錄（Whisper）」的模型選單選擇即可，
   第一次選新模型時會自動下載（需要網路）。

Q：找不到 API Key 設定欄位？
A：v3.4 起所有設定都集中在右上角「⚙ 設定」按鈕（快捷鍵 Cmd+,）。
